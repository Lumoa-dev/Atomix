## 二、设计问题修复

### 4. 钩子系统规模说明（附录/钩子与IS值参考.md）
- 添加设计说明段落：钩子分类为核心钩子（VM 原生，~15 个）和扩展钩子（编译器/库层实现），扩展钩子编译为条件跳转链不增加 VM 复杂度
- 每个钩子标注实现层级（P0/P1/P2）
- 在总纲中补充说明：轻量指二进制和内存，不指功能数量

### 5. $ 管道变量语法定位（通用语法.md §9）
- 在 §9 开头添加：`$` 是内置特殊变量，属于 `<标识符>` 的语法例外——它是唯一的单字符非字母标识符
- 在 §3 标识符定义中添加例外说明：`$` 是保留特殊标识符

### 6. `::` 分隔符消歧（区外语法.md）
- 在 §2.2 选择性导入末尾添加说明：`FROM ... USE ... :: Target` 中的 `::` 是导入语法专用分隔符，与跨域引用 `TOOLS :: func` 的 `::` 共享同一符号但上下文互斥——导入出现在文件头（所有域声明之前），跨域引用出现在域声明之后

### 7. NOT/NEG 编码模板（02-指令集规范.md）
- 将 NOT(0x2A) 和 NEG(0x2B) 的编码模板从 R3 改为 R1I
- 操作描述更新：`rd ← ~rs1` → `rd ← ~rd`（NOT 自操作），`rd ← -rd`（NEG 自操作）
- 添加设计说明：unary 指令使用单寄存器模板节省编码空间

### 8. MCPY/MSET 参数顺序说明（02-指令集规范.md）
- §3.8 表格中操作列改为：`memcpy(dst=rd, src=rs1, len=rs2)` 和 `memset(dst=rd, val=rs1, len=rs2)` 显式标注参数名
- 添加注释说明 rd 承载 dst 而非传统 memcpy 第一个参数位置的原因

### 9. DB 数据源参数补全（附录/数据源地址与参数速查.md）
- 将"待定"改为具体参数：`query`（查询语句）、`params`（参数绑定）、`timeout`（超时）、`driver`（驱动选择）

### 10. WEBS 命名说明（INPUT语法.md §6.1）
- 在 WEBS 行添加注释：`WEBS` = Web + Socket 的缩写，涵盖所有基于 Socket 的网络通信（TCP/HTTP/WebSocket 等），非"Web 专用"
