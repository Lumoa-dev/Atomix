# Atomix 文档图表统一方案

## 总体策略
将文档中的 ASCII 图分三类处理：
- **🅰️ 流程/状态/架构类 (~24处)** → Mermaid 代码块
- **🅱️ 位域/空间排布/对比类 (~8处)** → Python 生成 PNG → `.assets/`
- **🅲 目录树/代码对照 (~15处)** → 保留原样

## 实施步骤

### Phase 1：扩展 generate_charts.py
新增 4 个函数生成 8 张 PNG：
- `chart_instruction_encodings()` — 5 张位域图（总格式 + R3/R2I/R1I/JI）
- `chart_memory_watermark()` — 内存三区段水位线图
- `chart_vm_architecture()` — VM Runtime 架构总览图
- `chart_config_comparison()` — 三种配置方案对比图

运行一次生成所有 🅱️ 类 PNG 到 `.assets/`。

### Phase 2：逐文件 Mermaid 替换
分 6 批按文件执行，每个 ASCII 图替换为等效 Mermaid：
1. `INPUT语法.md`、`OUT语法.md`、`通用语法.md`、`类型系统.md`、`区外语法.md`
2. `编译管线.md`、`包管理.md`、`配置设计.md`
3. `编译行为.md`、`执行器设计.md`
4. `01-总纲与哲学.md`
5. `策略模块.md`、`02-指令集规范.md`
6. `运行时架构.md`

### Phase 3：验证
- 逐文件检查 Mermaid 语法
- 确认图片引用路径正确
- `git diff --stat` 确认改动范围
