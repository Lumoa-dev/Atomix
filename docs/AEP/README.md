# Atomix Enhancement Proposals (AEP)

AEP 是 Atomix 语言和运行时的设计扩展提案系统。
每个 AEP 编号递增，以 Draft 状态提交，经评审后推进到 Accepted / Implemented / Deferred。

## 提案流程

1. 发现问题或设计缺口
2. 按 `000-模板.md` 格式编写 AEP
3. 合并为 Draft 状态
4. 评审讨论
5. 推进到 Accepted（批准实施）或 Deferred（暂缓）
6. 实施完成后标记为 Implemented
