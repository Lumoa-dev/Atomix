## 域关键词索引

### 区外（文件级，区域声明之前）
`USE` `FROM` `EXCEPTION` `enum` `type` `TEST`

### TOOLS 域
`TOOLS` `fn` `PUB` `return` `do`

### INPUT 域
`INPUT` `WEBS` `FILES` `MEMS` `HTTP` `TCP` `DB` `OSS` `TXT` `CSV` `JSON` `JSONS` `YAML` `TOML` `XML` `JOIN`

### WORKS 域
`WORKS` `fn` `PUB` `return` `do` `IF` `ELIF` `ELSE` `FOR` `BREAK` `CONTINUE` `CALL`

### TASK 域
`TASK` `CALL` `IF` `ELIF` `ELSE` `FOR` `BREAK` `CONTINUE` `WAIT` `GOOUT`

### OUT 域
`OUT` `HTTP` `TCP` `FILES` `DB` `OSS`

### 通用（所有域）
`ASSERT` `RAISE` `CONST` `and` `or` `not` `true` `false` `$`
