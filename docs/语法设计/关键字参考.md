## 使用说明

本文档是纯参考手册，按关键字字母序排列。每个条目包含：
- 方向（INPUT / OUT / 双方向）
- 参数表
- 行为说明
- 示例

## 关键字列表

### `BIN`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 读二进制文件（等价 FILES） |
| OUT 行为 | 写二进制文件（等价 FILES） |
| 参数 | `encoding`、`mode`(overwrite/append) |

---

### `CALL`

| 属性 | 说明 |
|------|------|
| 使用区 | TASK |
| 概述 | 函数调用 |
| 模式 | `=>` / `=` / `=> OUT :` / `\/` |

详见《TASK语法.md》第 2 节。

---

### `CSV`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 按行解析为 `list[dict]` |
| OUT 行为 | 序列化后写入 |

**INPUT 参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `delimiter` | `string` | `,` | 分隔符 |
| `has_header` | `bool` | `true` | 是否首行为列名 |
| `encoding` | `string` | — | 编码 |

**OUT 参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `delimiter` | `string` | `,` | 分隔符 |
| `encoding` | `string` | — | 编码 |
| `mode` | `string` | `overwrite` | overwrite / append |

---

### `ELSE`

| 属性 | 说明 |
|------|------|
| 使用区 | TASK |
| 概述 | 与 `IF` 搭配，构成条件分支的否定分支 |

详见《TASK语法.md》第 4 节。

---

### `FILES`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 读原始文件字节 |
| OUT 行为 | 写原始文件字节 |

**INPUT 参数：** `encoding`、`mode`  
**OUT 参数：** `mode`（overwrite/append）、`encoding`

---

### `HTTP`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 发起请求，获取响应体 |
| OUT 行为 | 发送请求体到目标 |

**INPUT 参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `method` | `string` | `GET` | GET / POST / PUT / DELETE / PATCH |
| `header` | `string` | — | 请求头，可多条 |
| `body` | `string` | — | 请求体（POST/PUT 时有效） |
| `timeout` | `int` | `30` | 超时秒数 |
| `version` | `string` | `1.1` | HTTP 版本 |
| `ssl` | `bool` | `false` | 启用 SSL |

**OUT 参数：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `method` | `string` | `POST` | 请求方法 |
| `header` | `string` | — | 请求头 |
| `timeout` | `int` | `30` | 超时秒数 |
| `retry` | `int` | `0` | 重试次数 |

**示例：**

```
// INPUT
HTTP : "https://api.com/data" (method=GET) => RAW : bytes

// OUT
RAW => HTTP : "https://api.com/upload" (method=POST)
```

---

### `HTTPS`

与 HTTP 完全一致，默认 `ssl=true`。

---

### `IF`

| 属性 | 说明 |
|------|------|
| 使用区 | TASK |
| 概述 | 条件分支。支持数值比较和状态判断两种条件，可搭配 `ELSE` 使用 |

详见《TASK语法.md》第 4 节。

---

### `INPUT`（关键字）

| 属性 | 说明 |
|------|------|
| 使用区 | TASK, OUT |
| 概述 | 引用 INPUT 区定义的文件级常量 |
| 写法 | `INPUT : <常量名>`，如 `INPUT : APP_CFG` |

---

### `JOIN`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + TASK |
| 行为 | 多元聚合为一。将多个来源聚合为单一变量。INPUT 区用于声明时聚合，TASK 区用于调用时聚合 |

详见：
- INPUT JOIN：《IO语法.md》第 3 节
- TASK JOIN：《TASK语法.md》第 6 节

---

### `JSON`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 解析为结构化数据 |
| OUT 行为 | 序列化后写入 |

**INPUT 参数：** `encoding`、`schema`（可选校验文件）  

**OUT 参数：** `encoding`、`indent`、`mode`

---

### `MEM`

| 属性 | 说明 |
|------|------|
| 方向 | INPUT + OUT |
| INPUT 行为 | 只读拷贝，不暴露指针 |
| OUT 行为 | 只写，不可读回 |
| 参数 | `len`（长度）、`offset`（偏移） |

**示例：**

```
MEM : 0x7ffd0000 (len=4096) => BOOT : bytes
HEADER => MEM : 0x7ffd1000 (len=256)
```

---
