# 附录 A：数据源地址与参数速查

> 配套文档: 详见 INPUT语法.md

---

## A.1 WEBS（网络基础）

| 项目 | 说明 |
|------|------|
| 地址格式 | `"host:port"` 或 `"scheme://host:port"` |
| 参数 | 待定（超时、重连策略等） |
| 产出类型 | `bytes` |

---

## A.2 TCP

| 项目 | 说明 |
|------|------|
| 基于 | WEBS |
| 地址格式 | `"host:port"` |
| 参数 | `timeout`：超时时间 |
| 产出类型 | `bytes` |

---

## A.3 HTTP

| 项目 | 说明 |
|------|------|
| 基于 | WEBS |
| 地址格式 | `"https://host[:port]/path"` 或 `"http://host[:port]/path"` |
| 参数 | `method`：请求方法（GET/POST/PUT/DELETE 等），默认 GET |
| | `timeout`：超时时间 |
| | `headers`：请求头 |
| | `body`：请求体（用于 POST/PUT） |
| 协议支持 | HTTP/1.1、HTTP/2，前瞻 HTTP/3 |
| 产出类型 | `bytes` |

---

## A.4 DB

| 项目 | 说明 |
|------|------|
| 基于 | WEBS |
| 地址格式 | 连接字符串，格式由具体数据库驱动决定 |
| 参数 | `query`：查询语句或表名 |
| | `params`：参数绑定（列表或字典） |
| | `timeout`：查询超时 |
| | `driver`：驱动选择（如 `"mysql"`、`"postgres"`、`"sqlite"`） |
| 产出类型 | 行集/记录集 |
| 说明 | 通用接口，各数据库差异由驱动层处理 |

---

## A.5 OSS

| 项目 | 说明 |
|------|------|
| 基于 | WEBS |
| 地址格式 | `"endpoint/bucket/key"` |
| 参数 | `access_key`、`secret_key` 等身份信息 |
| 产出类型 | `bytes` |
| 说明 | 兼容 S3、阿里 OSS、MinIO 等 |

## A.6 FILES（文件基础）

| 项目 | 说明 |
|------|------|
| 地址格式 | 操作系统文件路径 |
| 参数 | `encoding`：字符编码（默认 UTF-8） |
| 产出类型 | `bytes` |
| 说明 | 通读任何文件，裸字节输出。内容解析交给衍生关键词或用户自行处理 |

## A.14 MEMS（内存基础）

| 项目 | 说明 |
|------|------|
| 地址格式 | 内存地址 / 句柄名 / 抽象标识符 |
| 参数 | `offset`：偏移量（默认 0） |
| | `size`：读取大小（默认由实现决定） |
| 产出类型 | `bytes` |
| 说明 | 纯内存寻址，地址格式由 MEMS 实现层具体定义 |
