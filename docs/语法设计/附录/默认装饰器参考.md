# 附录 B：默认装饰器参考

> 装饰器是挂接在数据流箭头 `=>` 上的函数，处理流经的数据。
> 装饰器是**通用设施**——不限定区域，能用处自然能用。此处仅列出默认提供的装饰器。
> 默认装饰器由 TOOLS 区预定义，用户可直接使用，无需额外声明。
> 配套文档: [INPUT语法.md](../INPUT语法.md)、[通用语法.md](../通用语法.md)

---

## B.1 编解码

| 装饰器 | 说明 |
|--------|------|
| `[encode]` | 编码（默认 UTF-8） |
| `[decode]` | 解码（默认 UTF-8） |
| `[base64]` | Base64 编码/解码 |
| `[hex]` | 十六进制编解码 |
| `[urlencode]` | URL 编码 |

## B.2 压缩

| 装饰器 | 说明 |
|--------|------|
| `[gzip]` | Gzip 压缩/解压 |
| `[deflate]` | Deflate 压缩/解压 |
| `[zlib]` | Zlib 压缩/解压 |

## B.3 加密与安全

| 装饰器 | 说明 |
|--------|------|
| `[encrypt]` | 加密（算法由参数指定） |
| `[decrypt]` | 解密 |
| `[hash]` | 哈希（md5/sha1/sha256 等） |
| `[sign]` | 签名 |
| `[verify]` | 验签 |

## B.4 验证

| 装饰器 | 说明 |
|--------|------|
| `[validate]` | 数据格式验证（JSON Schema 等） |
| `[check]` | 条件检查，不通过则中断 |
| `[assert]` | 断言，失败抛异常 |

## B.5 数据转换

| 装饰器 | 说明 |
|--------|------|
| `[to_json]` | 转为 JSON 字符串 |
| `[to_yaml]` | 转为 YAML 字符串 |
| `[to_csv]` | 转为 CSV 格式 |
| `[flatten]` | 展平嵌套结构 |
| `[filter]` | 按条件过滤数据 |
| `[map]` | 逐元素映射转换 |

## B.6 流量控制

| 装饰器 | 说明 |
|--------|------|
| `[retry]` | 失败重试（次数/间隔可配置） |
| `[timeout]` | 超时控制 |
| `[throttle]` | 限流（每秒/每分钟最大次数） |
| `[cache]` | 缓存结果（TTL 可配置） |

---

> 装饰器可串联使用：`[gzip] [encrypt] =>`
> 
> 此列表参考 Python 标准库及生态中常见的装饰器模式裁剪而来。后续可按需增补。
