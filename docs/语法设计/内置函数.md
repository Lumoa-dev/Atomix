# Atomix 内置函数

> 架构版本: v0.1 (设计阶段)
> 适用范围: 所有语法域，无需导入即可使用

---

## 1. 设计原则

内置函数覆盖**除 IO 以外的全部通用操作**。IO 操作由 INPUT/OUT 区及数据源关键词处理。

所有内置函数可在任何语法域中直接使用，无需 `USE` 导入。

---

## 2. 类型转换

| 函数 | 说明 |
|------|------|
| `int(x)` | 转整数 |
| `float(x)` | 转浮点 |
| `str(x)` | 转字符串 |
| `bytes(x)` | 转字节序列 |
| `bool(x)` | 转布尔 |

## 3. 集合操作

| 函数 | 说明 |
|------|------|
| `len(x)` | 返回长度（字符串、列表、字典等） |
| `range(n)` | 生成整数序列 `[0, n)` |
| `range(start, end)` | 生成 `[start, end)` 序列 |

## 4. 字符串操作

| 函数 | 说明 |
|------|------|
| `split(s, sep)` | 分割字符串 |
| `join(list, sep)` | 拼接字符串列表 |
| `contains(s, sub)` | 是否包含子串 |
| `replace(s, old, new)` | 替换子串 |

## 5. 数学运算

| 函数 | 说明 |
|------|------|
| `abs(x)` | 绝对值 |
| `min(a, b)` | 取小值 |
| `max(a, b)` | 取大值 |

## 6. 调试与内省

| 函数 | 说明 |
|------|------|
| `print(x)` | 打印输出 |
| `type(x)` | 返回值的运行时类型 |
| `assert(cond, msg)` | 断言（通用语法 §12 已定义 `ASSERT`，`assert` 为函数形式） |

---

> 内置函数清单在实现阶段可根据需要增补。IO 部分由 INPUT/OUT 区及数据源关键词覆盖，不在此列。
