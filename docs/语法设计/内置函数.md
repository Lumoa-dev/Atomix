# Atomix 内置函数

> 架构版本: v0.1 (设计阶段)
> 最后更新: 2026-07-15
> 所属体系: 语法设计

---

## 总览

内置函数是编译器直接提供的函数，不依赖 TOOLS 定义，不依赖宿主语言。全区可用（INPUT / TASK / OUT / WORKS / TOOLS 均可使用）。

**设计约束：** 内置函数仅限于纯数学运算和纯数据操作。不提供任何文件操作、网络操作、I/O 相关函数——那些归 INPUT/OUT 区的关键字管。

---

## 1. 类型转换

### to_string

```
to_string(<值>) -> string
```

将值转换为字符串表示。

```
string s = to_string(42)        // "42"
string t = to_string(true)      // "true"
```

### to_int

```
to_int(<值>) -> int
```

转换为整型。输入为字符串时尝试解析，浮点数截断。

```
int i = to_int("42")            // 42
int j = to_int(3.14)            // 3（截断）
```

### to_float

```
to_float(<值>) -> float
```

转换为浮点型。

```
float f = to_float(42)          // 42.0
```

### to_bytes

```
to_bytes(<值>) -> bytes
```

转换为字节序列。

```
bytes b = to_bytes("hello")     // 0x68656C6C6F
```

## 2. 数学运算

### abs

```
abs(<数值>) -> <数值>
```

绝对值。

```
int n = abs(-42)                // 42
float m = abs(-3.14)            // 3.14
```

### floor

```
floor(<数值>) -> int
```

向下取整。

```
int n = floor(3.14)             // 3
int m = floor(-3.14)            // -4
```

### ceil

```
ceil(<数值>) -> int
```

向上取整。

```
int n = ceil(3.14)              // 4
int m = ceil(-3.14)             // -3
```

### round

```
round(<数值>) -> int
```

四舍五入。

```
int n = round(3.14)             // 3
int m = round(3.86)             // 4
```

### sqrt

```
sqrt(<数值>) -> float
```

平方根。

```
float s = sqrt(9.0)             // 3.0
```

### pow

```
pow(<底数>, <指数>) -> float
```

幂运算。

```
float p = pow(2.0, 10.0)        // 1024.0
```

### sin / cos / tan

```
sin(<弧度>) -> float
cos(<弧度>) -> float
tan(<弧度>) -> float
```

三角函数。

```
float s = sin(3.14159 / 2)      // ~1.0
```

---

## 3. 数据检视

### len

```
len(<值>) -> int
```

获取数据的长度/大小。支持类型：string、bytes、list、dict。

```
int n = len("hello")            // 5
int m = len([1, 2, 3])          // 3
```

### type_of

```
type_of(<值>) -> string
```

返回值的类型名称（字符串）。

```
string t = type_of(42)           // "int"
string u = type_of("hello")     // "string"
```

---

## 4. 比较

### min

```
min(<值>, <值>) -> <值>
```

返回两个值中的较小者。

```
int m = min(3, 7)               // 3
```

### max

```
max(<值>, <值>) -> <值>
```

返回两个值中的较大者。

```
int m = max(3, 7)               // 7
```

---

## 5. 集合

### range

```
range(<end>) -> list
range(<start>, <end>) -> list
range(<start>, <end>, <step>) -> list
```

生成整数序列，与 `for` 搭配使用。

```
for (i in range(5)) { ... }         // 0, 1, 2, 3, 4
for (i in range(2, 5)) { ... }      // 2, 3, 4
for (i in range(0, 10, 2)) { ... }  // 0, 2, 4, 6, 8
```

---

## 6. 输出

### print

```
print(<值>)
print(<格式>, <值>, ...)
```

输出到标准输出。仅在开发模式有效，生产模式可被编译器优化移除。

```
print("hello world")
print("count=%d, name=%s", count, name)
```

> `print` 是纯输出操作，不涉及文件 I/O 或网络 I/O。它不写入文件、不发送请求、不持久化——只是把值投递给标准输出流。
