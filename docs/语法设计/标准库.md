# Atomix 标准库

> 架构版本: v0.1 (设计阶段)
> 适用范围: 所有语法域（按模块分别导入）
> 配套文档: 详见 包管理.md、内置函数.md

---

## 1. 设计原则

标准库覆盖编写实际任务所需的**通用功能**，按模块组织。每个模块通过 `USE` 导入：

```
USE : "json"
USE : "log"
USE : "crypto"
```

- 不导入不加载——零开销
- 纯 TOOLS 函数——无副作用，可用于装饰器位置
- 所有函数在编译为目标平台时通过 ECALL 或编译器内联实现

---

## 2. json — JSON 解析与序列化

```
USE : "json"
```

| 函数 | 签名 | 说明 |
|------|------|------|
| `json_parse` | `fn json_parse<T>(raw : bytes) : T` | 将 JSON 字节解析为 Atomix 结构体。`T` 由调用上下文推断 |
| `json_stringify` | `fn json_stringify(v : any) : str` | 将 Atomix 值序列化为 JSON 字符串 |
| `json_valid` | `fn json_valid(raw : bytes) : bool` | 检查是否为合法 JSON |

```
x = json_parse(INPUT :: RAW)          # RAW 是 bytes，解析为对应类型
s = json_stringify({a: 1, b: 2})      # → '{"a":1,"b":2}'
```

## 3. log — 结构化日志

```
USE : "log"
```

| 函数 | 签名 | 说明 |
|------|------|------|
| `log_info` | `fn log_info(msg : str)` | 信息级日志 |
| `log_warn` | `fn log_warn(msg : str)` | 警告级日志 |
| `log_error` | `fn log_error(msg : str)` | 错误级日志 |
| `log_debug` | `fn log_debug(msg : str)` | 调试级日志（仅 dev 模式输出） |

日志自动附带 Step 名、任务 ID、时间戳。不需要手动拼接上下文：

```
log_info("starting data pipeline")
log_error(f"fetch failed: {ISERROR}")
```

## 5. math — 数学扩展

内置函数（详见 内置函数.md）已覆盖 `abs`、`min`、`max`。标准库补充：

```
USE : "math"
```

| 函数 | 签名 | 说明 |
|------|------|------|
| `math_sqrt` | `fn math_sqrt(x : float) : float` | 平方根 |
| `math_pow` | `fn math_pow(base : float, exp : float) : float` | 幂运算 |
| `math_sin` | `fn math_sin(x : float) : float` | 正弦（弧度） |
| `math_cos` | `fn math_cos(x : float) : float` | 余弦（弧度） |
| `math_floor` | `fn math_floor(x : float) : int` | 向下取整 |
| `math_ceil` | `fn math_ceil(x : float) : int` | 向上取整 |
| `math_round` | `fn math_round(x : float) : int` | 四舍五入 |
| `math_log` | `fn math_log(x : float) : float` | 自然对数 |
| `math_log2` | `fn math_log2(x : float) : float` | 以 2 为底对数 |
| `math_log10` | `fn math_log10(x : float) : float` | 以 10 为底对数 |

> 三角函数和超越函数通过宿主 libm 实现，走 ECALL。

## 6. time — 时间与计时

```
USE : "time"
```

| 函数 | 签名 | 说明 |
|------|------|------|
| `time_now` | `fn time_now() : str` | 当前 UTC 时间（ISO 8601 格式） |
| `time_unix` | `fn time_unix() : int` | 当前 Unix 时间戳（秒） |
| `time_sleep` | `fn time_sleep(d : duration)` | 暂停执行 |

```
start = time_unix()
log_info(f"pipeline started at {time_now()}")
time_sleep(5s)
elapsed = time_unix() - start
```
